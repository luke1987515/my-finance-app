# my-finance-app

# 📊 人生花費結構：預算 vs 實際支出 終極儀表板

這是一個基於 Python 與 Streamlit 開發的互動式個人財務規劃工具。
它可以幫助使用者動態調整人生的預算比例，並與實際生活支出進行視覺化對比，找到潛在的財務漏洞。

👉 **[點此線上使用我的 App](https://luke1987515-my-finance-app.streamlit.app)**

---

## 🚀 核心功能

### 1. 👈 100% 自動平衡預算拉桿
* 採用**數學歸一化**邏輯：當你手動調大某一項花費比例時，其他項會「按原本比例自動縮小」，永遠維持總和為 **100%**。
* 輸入月薪後，系統會自動將比例轉換為每月或每年的真實金額，讓預算更有真實感。

### 2. 👉 智慧防呆實際支出填寫
* 提供**大類別 ➡️ 子項目**的連動下拉選單。
* 具備「時間週期」自動換算魔法：不論你輸入每日、每週、每月或每年，系統一律在背景統一換算成「每月金額」進行對比。

### 3. 📊 雙甜甜圈圖動態對比
* 底部提供「理想預算」與「實際支出」的圓餅圖。
* 支援「每月金額」與「每年金額」一鍵雙視角切換。

## 🛠️ 開發與使用技術

* **開發語言**：Python 3.12.3
* **前端介面**：Streamlit (相容 2026 年最新版本)
* **視覺化圖表**：Plotly Express (提供精美流暢的動態甜甜圈圖)
* **數據處理**：Pandas

## 💻 本地端快速啟動

如果你想在自己的電腦上執行這個專案，請跟著以下步驟：

1. **克隆（Clone）或下載此專案**：
   ```bash
   git clone https://github.com/luke1987515/my-finance-app.git
   ```

2. **安裝所需套件**：
   ```bash
   pip install -r requirements.txt
   ```

3. **啟動 App**：
   ```bash
   streamlit run app_final.py
   ```
